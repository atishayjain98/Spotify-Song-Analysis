# Spotify-Song-Analysis

This project performs EDA on a large spotify dataset containing tracks and artists information.
The objective is to uncover data insights related to -
- Music Trends
- Characterstics of Audio features
- Song Popularity
- Relationship between tracks and artists

# Goals 

- Load and Explore data structure
- Clean and transform data
- Convert and engineer relevant features
- Visualize features, correlation and trends
- Generate meaningful insights
